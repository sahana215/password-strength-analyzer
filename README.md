#  Password Strength Analyzer

A Password Strength Analyzer built using **Python** in **Visual Studio Code**. This application checks the strength of a user-entered password based on security rules and provides suggestions to improve password security.

##  Features

- Checks minimum password length
- Detects uppercase letters
- Detects numeric characters
- Detects special characters
- Identifies common weak passwords
- Classifies password as Weak, Medium, or Strong
- Estimates password crack time
- Provides suggestions for creating a stronger password

##  Technologies Used

- Python 3
- Visual Studio Code

##  How to Run

1. Open the project in Visual Studio Code.
2. Open the integrated terminal.
3. Run the following command:

```bash
python password_checker.py
```
