# Password Strength Analyzer 
This project was developed as part of my Cyber Security Internship at THIRANEX.
##  Project Description
The Password Strength Analyzer checks whether a password is weak, medium, or strong based on different security criteria.
##  Features
- Password length check
- Uppercase letter detection
- Number detection
- Special character detection
- Common password detection
- Security suggestions
## Technologies Used
- Python
- Regex (re module)
- VS Code
## ▶️ How to Run

```bash
python password_checker.py
```
## 📷 Sample Output
```text

Password Analysis Result
Strength: Medium Password
Estimated crack time: Few Days
```
